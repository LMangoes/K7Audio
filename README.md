# K7/Audio



Local-library music player.



Base: Electron + vanilla JS, no cloud dependency, no telemetry.





All app state ('covers', 'custom-tags', 'library-cache', 'playlists', 'settings') is written to `..\\K7Audio\\data\\`. Moving or copying the whole `K7Audio` folder (e.g. to a different drive) and playlists/settings moves with it.
